#### Install

在之前，也许你会有兴趣了解下[Muggle Cli](https://github.com/isunyuan/muggle-cli)，通过脚手架的方式创建。

```shell
// clone
git clone https://github.com/isunyuan/react-template.git
cd react-template
yarn
yarn start
```
